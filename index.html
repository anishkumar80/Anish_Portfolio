Projects

Library Management System

The Library Management System is a command-line application designed to streamline the process of managing library resources. It allows users to search, issue, return for books, check availability, and manage their accounts. The system includes features for librarians to add new books, delete books, update book information, manage Admin & Users, and generate reports on library usage. The goal is to create an efficient and user-friendly platform for both library Admins and Users.

Face Detection

The Face Detection project is a Python-based application that utilizes OpenCV and Haar Cascades to detect faces in images and real-time video streams. The program captures video from the webcam, processes each frame to identify faces, and draws rectangles around detected faces. This project demonstrates the use of computer vision techniques for face detection, which can be applied in various fields such as security, user authentication, and human-computer interaction.

Employee Management Record System

The Employee Management Record System is a command-line application designed to manage employee records efficiently. It allows HR personnel to add, update, and delete employee information, track attendance, and generate reports. The system aims to streamline HR processes and improve data accuracy and accessibility.

Cafe Management System

The Cafe Management System is a command-line application designed to manage cafe operations efficiently. It allows users to place orders, manage inventory, and track sales. The system aims to enhance the customer experience and improve operational efficiency.
